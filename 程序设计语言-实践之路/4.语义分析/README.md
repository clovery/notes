> 语法关注的是合法程序的形式，而语义关注它们的意义。

如 1 + 2，前者关注形式，后者关注意义。

> 语言语法关注的就是语言定义中可以用上下文无关法方便描述的那一部分，而语义就是定义中不能这样做的那一部分。

## 4.1 语义分析器所扮演的角色

> 语义分析器创建起一棵带标注的语法树，随后由中间代码生成器将它翻译成一种线性形式。

断言捕捉程序本身的错误。

> 一个断言就是一个语句，它描述了当执行到达代码中的某点时期望为真的一个条件。
