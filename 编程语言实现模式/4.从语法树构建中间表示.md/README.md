## 4.2 构建抽象语法树 (Building Abstract Syntax Trees)

理想的 IR 树应该有如下性质:

* 紧凑: 不含无用的节点

* 易用: 很容易遍历

* 显意： 突出操作符、操作对象，以及它们相互间的关系，不拘泥于文法中的东西
